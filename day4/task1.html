Html

HTML stands for HyperText Markup Language.It is the standard markup language used to create web pages.
  1. Function of html:Structures content on the web.Uses elements represented by tags.
  2. Creation of html:Created by Tim Berners-Lee.Year of creation: 1991
  3. Purpose of html:Initially designed to allow researchers to share documents via the internet.Facilitated communication between scientists and researchers.
  4. Impact of html :Revolutionized the way information is shared and accessed globally.
  5. Main Keywords:html,head,body

Heading Tags:

h1: Defines the most important heading, typically the main title.
h2: Defines a secondary heading, used for major sections.
h3: Defines a tertiary heading, used for subsections under h2.
h4: Defines a quaternary heading, used for smaller sections.
h5: Defines a fifth-level heading, used for even smaller sections.
h6: Defines the least important heading, used for the smallest sections.

Paragraph Tag:

The paragraph tag in HTML is used to define and format a block of text as a paragraph.

Fomating Tags:

Formatting tags in HTML define the structure and style of content on webpages, ensuring proper display and presentation.
  1. Bold:Makes text bold.
  2. Italic:Makes text italicized
  3. Strong:Indicates strong importance, typically displayed as bold.
  4. Underline: Underlines text.
  5. Subscript:Renders text as subscript.
  6. Superscript:Renders text as superscript.
  7. Delete:Represents deleted or removed text
  8. Mark:Highlights text for reference or notation.
  9. Blockquote: Indicates a block of quoted text.
  10. Abbreviation: Defines an abbreviation or acronym.
  11. Quote: Defines a short inline quotation
  12. Preformatted: Preformats text, maintaining whitespace and line breaks.
  13. Cite: Defines the title of a work being cited.
  14. Address: Defines contact information.
  15. Code: Represents a fragment of computer code.
  16. Keyboard Input: Indicates keyboard input.
  17. Emphasis : Emphasizes text typically rendered in italics.

Table Tag:

The table tag in HTML is used to create a structured grid layout for organizing data into rows and columns. It allows for the creation of tables with headers, body content, and footers, facilitating organized presentation of tabular data on webpages.

Table Tag Features:

1.Table Caption: Adds a title or description to the table
2.Cellpadding: Specifies space between the cell content and cell border.
3.Cellspacing: Specifies space between cells.
4.Bgcolor: Sets the background color of the table or specific cells.
5.Width and Height: Defines the dimensions of the table.
6.Grouping: Allows thead, tbody, and tfoot tags to group table rows.
7.Spanning: Uses colspan and rowspan attributes to span multiple columns or rows, respectively.

Absolute Path:

An absolute path specifies the full address of a file or resource starting from the root directory or domain, ensuring an exact location.
Relative Path:
A relative path specifies the location of a file or resource relative to the current directory or context, not starting from the root.

Inline level element:

These elements do not start on a new line and only take up as much width as necessary, such as span, a, and img.

Block level element:

These elements start on a new line and take up the full width available, such as div, p, and h1.

Tag:

The anchor tag in HTML is used to create hyperlinks, which allow users to navigate to other web pages, resources, or locations within the same page. It uses the href attribute to specify the URL or the target location of the link.

Anchor Tag Attributes:

href: Specifies the URL of the page or resource the link goes to.
target: Specifies where to open the linked document
title: Provides additional information about the link, often displayed as a tooltip.
download: Indicates that the linked resource should be downloaded instead of opened.

Image Tag:

The img tag is used to embed an image in an HTML page.Images are not technically inserted into a web page; images are linked to web pages. The img tag creates a holding space for the referenced image.

Image Tag Attributes:

src: Specifies the path to the image file.
alt: Provides alternative text for the image, crucial for accessibility.
width: Sets the width of the image.
height: Sets the height of the image.
title: Adds a tooltip text that appears when the mouse hovers over the image.
